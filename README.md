# MQespol
mQespol is app MQTT, We can recieve data from devices IoT, store and visualize it.

---

Design Document  

Ronaldo Rodriguez 
## Introduction 
Have you ever seen a app that recolet data without internet from devices IoT?
MQespol can help you to do that.
Features:
-	Recolet Data in a Local Area Network
-	Easy way to add a Device IoT for sharing data.
-	 Send Data to a server remote my MQTT protocol.
-	Visualize the data anywhere with Node-Red with Grafana.  

Use your Android device to create your own recoleter data from differentes devices Iot connected to your Local network.
